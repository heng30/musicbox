#### 简介
这是一个专门为安卓开发的音乐播放器。基于Flutter和Rust。

#### 功能
- [x] 播放本地音乐
- [x] 管理播放列表
- [x] 搜索和下载bilibili音频文件
- [x] 搜索，预览和下载歌词
- [x] 调整歌词播放速度
- [x] 本地备份/恢复数据库和配置

#### 如何构建？
- 安装 `Cargo` 和 `FVM`
- 运行 `make` 构建安卓发布版本
- 运行 `make debug` 调试
- 参考 [Makefile](./Makefile) 了解更多信息
